# 🚀 一键启动说明

## 📁 启动脚本文件

项目根目录包含以下启动脚本：

1. **启动工具.bat** - Windows批处理脚本（推荐）
2. **启动工具.ps1** - PowerShell脚本（功能更强大）
3. **停止服务.bat** - 停止所有服务

---

## 🎯 使用方法

### 方法1：双击启动（最简单）

1. 在文件资源管理器中打开：`F:\ProReport-GenAI`
2. 双击 **`启动工具.bat`**
3. 等待服务启动完成
4. 自动打开浏览器访问：http://localhost:3000

### 方法2：使用PowerShell脚本

1. 右键点击 **`启动工具.ps1`**
2. 选择"使用PowerShell运行"
3. 如果提示执行策略限制，运行：
   ```powershell
   Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
   ```

### 方法3：命令行启动

```bash
cd F:\ProReport-GenAI
启动工具.bat
```

---

## ✨ 脚本功能

### 自动检查
- ✅ 检查Node.js和npm是否安装
- ✅ 检查依赖是否已安装（未安装则自动安装）
- ✅ 检查环境配置文件（不存在则创建默认配置）

### 自动启动
- ✅ 同时启动前端（端口3000）和后端（端口3001）
- ✅ 显示启动状态和访问地址
- ✅ 彩色输出，易于查看

---

## 🛑 停止服务

### 方法1：使用停止脚本
双击 **`停止服务.bat`**

### 方法2：在启动窗口中
按 `Ctrl + C` 停止服务

### 方法3：任务管理器
1. 按 `Ctrl + Shift + Esc` 打开任务管理器
2. 找到 `node.exe` 进程
3. 右键 → 结束任务

---

## 📋 启动后操作

1. **等待启动完成**
   - 看到 "VITE ready" 和 "Server running" 表示启动成功

2. **打开浏览器**
   - 访问：http://localhost:3000
   - 或点击脚本中显示的链接

3. **首次使用**
   - 点击"立即注册"创建账号
   - 注册后自动登录

---

## ⚠️ 注意事项

### 端口占用
如果端口3000或3001被占用：
- 关闭占用端口的程序
- 或修改配置文件中的端口号

### 防火墙提示
首次运行时，Windows防火墙可能会提示：
- 选择"允许访问"

### 权限问题
如果遇到权限问题：
- 右键脚本 → "以管理员身份运行"

---

## 🔧 故障排除

### 问题1：脚本无法运行
**解决：**
- 确保文件扩展名是 `.bat` 或 `.ps1`
- 检查文件是否被Windows Defender阻止

### 问题2：依赖安装失败
**解决：**
```bash
# 手动安装依赖
npm install
cd server
npm install
```

### 问题3：服务启动失败
**解决：**
1. 检查 `server/.env` 配置是否正确
2. 检查端口是否被占用
3. 查看错误信息，根据提示解决

## 💡 快捷方式

### 创建桌面快捷方式

1. 右键 **`启动工具.bat`**
2. 选择"发送到" → "桌面快捷方式"
3. 重命名为"启动ProReport"
4. 双击即可启动

### 固定到任务栏

1. 创建桌面快捷方式
2. 右键快捷方式 → "固定到任务栏"
