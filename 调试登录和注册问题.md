# 🔍 调试登录和注册问题

## 🐛 问题分析

1. **登录按钮点击无反应**
2. **没有看到注册按钮**

## 🔍 检查步骤

### 步骤1：检查浏览器控制台

1. **按 `F12` 打开开发者工具**
2. **切换到 "Console" 标签**
3. **查看错误信息**：
   - 是否有红色错误？
   - 是否有网络请求失败？
   - 是否有JavaScript错误？

### 步骤2：检查网络请求

1. **切换到 "Network" 标签**
2. **点击 "Login" 按钮**
3. **查看是否有请求发送**：
   - 应该看到 `/api/auth/login` 请求
   - 检查请求状态码
   - 检查请求URL是否正确

### 步骤3：检查注册按钮

注册按钮应该在登录表单底部，显示为：
- 文本："还没有账号？立即注册"
- "立即注册" 是一个可点击的链接（橙色）

如果看不到，可能是：
- 页面没有完全加载
- CSS样式问题
- 浏览器窗口太小（响应式设计）

## 🔧 快速修复方案

### 方案1：检查API调用

打开浏览器控制台（F12），在Console中输入：

```javascript
// 测试API地址
console.log('API Base URL:', window.location.origin + '/api');

// 测试登录请求
fetch('/api/auth/login', {
    method: 'POST',
    headers: {
        'Content-Type': 'application/json'
    },
    body: JSON.stringify({
        username: 'test',
        password: 'test123'
    })
})
.then(r => r.json())
.then(data => console.log('Response:', data))
.catch(err => console.error('Error:', err));
```

### 方案2：检查注册按钮

1. **滚动页面到底部**
2. **查看登录表单下方**
3. **应该看到 "还没有账号？立即注册"**

如果还是看不到，可能是CSS问题。

## 🆘 常见问题

### 问题1：登录按钮点击无反应

**可能原因**：
- JavaScript错误阻止了事件处理
- 表单验证失败
- API调用失败

**检查**：
- 浏览器控制台的错误信息
- Network标签中的请求状态

### 问题2：看不到注册按钮

**可能原因**：
- 页面没有完全加载
- CSS样式问题
- 浏览器窗口太小

**解决方案**：
- 刷新页面（`Ctrl + F5`）
- 检查页面是否完全加载
- 尝试滚动到底部

### 问题3：API请求失败

**可能原因**：
- API路由配置错误
- CORS问题
- 后端服务未启动

**检查**：
- Network标签中的请求状态
- 请求URL是否正确
- 响应内容是什么
