## 📖 用户使用说明

### 第一步：访问工具

1. 打开浏览器（推荐使用 Chrome、Edge 或 Firefox）
2. 访问：`https://proreport-genai.vercel.app`
3. 等待页面加载完成

### 第二步：注册账号

1. 在登录页面，点击底部的 **"立即注册"** 按钮（橙色按钮）
2. 填写注册信息：
   - **用户名**：至少3个字符（例如：`zhangsan`）
   - **密码**：至少6个字符（例如：`123456`）
   - **邮箱**：可选（例如：`zhangsan@example.com`）
3. 点击 **"Register"** 按钮完成注册
4. 注册成功后会自动登录

### 第三步：开始使用

1. **创建新项目**：
   - 点击 "新建项目" 按钮
   - 填写项目名称和描述
   - 选择使用标准模板或自定义大纲

2. **生成报告内容**：
   - 选择要生成的章节
   - 添加参考文件（格式参考、规范参考、内容参考等）
   - 点击 "生成" 按钮

3. **修改和优化**：
   - 选中要修改的文本
   - 输入修改指令
   - 点击 "修改" 按钮

### ⚠️ 重要提示

1. **数据安全**：
   - 每个用户的数据是独立的，不会互相干扰
   - 请妥善保管你的账号密码

2. **浏览器要求**：
   - 推荐使用现代浏览器（Chrome、Edge、Firefox 最新版本）
   - 如果遇到问题，请尝试清除浏览器缓存

3. **网络要求**：
   - 需要稳定的网络连接
   - 生成内容时可能需要等待一段时间

### ❓ 常见问题

**Q: 忘记密码怎么办？**
A: 目前不支持密码重置功能，请妥善保管密码。如果忘记密码，需要重新注册。

**Q: 可以多人使用同一个账号吗？**
A: 可以，但不推荐。建议每个用户注册自己的账号，以确保数据安全。

**Q: 数据会丢失吗？**
A: 数据存储在云端 MongoDB 数据库中，不会丢失。但建议定期导出重要项目。

**Q: 支持哪些文件格式？**
A: 支持 PDF、Word、Excel、图片等常见格式作为参考文件。

## 🔧 管理员信息（仅给你自己）

### 环境变量配置

确保在 Vercel 中设置了以下环境变量：
- `MONGODB_URI`：MongoDB Atlas 连接字符串（必需）
- `JWT_SECRET`：JWT 密钥（建议设置，提高安全性）
- `GEMINI_API_KEY`：Gemini API 密钥（必需，用于内容生成）

### 查看部署状态

1. 登录 Vercel 控制台
2. 进入项目 `proreport-genai`
3. 查看 "Deployments" 标签了解部署状态
4. 查看 "Logs" 标签查看运行时日志

### 更新工具

如果需要更新工具：
1. 修改代码
2. 提交到 GitHub
3. Vercel 会自动重新部署
